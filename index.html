Quem sou

Formado em Análise e Desenvolvimento de Sistema pela Anhanguera e hoje foco em Desenvolvimento Web com base na MERN Stack, abaixo cito algumas das tecnologias com as quais tenho contato, venho estudando e o quão fluente sou em cada uma delas.

Conhecimentos

• Geral

Familiaridade com diversos recursos que auxiliam no desenvolvimento de um site como Bancos de imagens, ilustrações, icones, fácil adaptação a tecnologias e dedicação para obter o resultado desejado.

• Figma

Edição, criação, modificação e componentização de grupos, noção de design e entendimento da padronização de componentes e onde certos elementos devem ficar, como aparecem e se comportam dependendo de onde estão na página. (com base no convivio e uso de outros sites)

• Inglês

Nível B1, boa compreensão com textos técnicos do stackoverflow.

Tecnologias

• HTML

Conhecimento das principais tags, criação de tabelas, headers, paragrafos, media, formulários, inputs, semântica e agilidade na criação de multiplas tags (EMMET Abbreviations)

• CSS

Conhecimento em Sass, principais seletores, alinhamento e criação de layout com Flex/Grid, conhecimento do comportamento de cada um dos positions, animações, criação de páginas responsivas com o uso de media queries, dentre outros.

• JavaScript

Criação e manipulação de elementos HTML e CSS, escopo, hoisting, familiaridade com Objetos/JSON e Arrays, destructuring, laços, callbacks, funções assíncronas, promises, consumo de API com fetch/axios, localStorage, cookies, sessionStorage, etc.

• Git

Principais comandos; add, status, commit, push, inserção de chave SSL, criação de branches e merges.

• NodeJS

Gerenciador de pacotes NPM, Cors, Criação de APIs com Express, consumo com Axios, paths, middleware, consumo e inserção de dados no mongoDB.

Objetivos

Aprender a trabalhar em grupo e fortalecer conhecimentos com novos desafios.
